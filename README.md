# Projet Personnel : IsYourPassStrong

**IsYourPassStrong** est un programme en Python qui permet de vérifier si un mot de passe est assez robuste selon sa longueur, les caractères utilisés, etc. et d'en générer.

Pour utiliser, la fonction qui permet de détecter si un mot de passe est évident, téléchargez le fichier _rockyou.txt.gz_ sur le site officiel. Dézippez-le et 
ajoutez dans le projet.

ℹ️ Ce projet est en cours de développement.
